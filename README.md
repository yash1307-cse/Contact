# Contact Manager
Contact Manager app in which user can add contacts, delete contacts. User also making calls from these app. 



## Features

- Light/dark mode supported
- Make calls
- Add contacts
- Delete contacts


## Tech Stack

**Architecture:** MVVM (Model - View - ViewModel)

**Language:**  Kotlin

**Database:**  Room
